# Translation Keys cho FilterDropdown Component

Các translation keys cần được thêm vào hệ thống i18n:

## Search Section
```json
{
  "nikki": {
    "general": {
      "search": {
        "title": "Tìm kiếm",
        "use_input": "Sử dụng ô tìm kiếm ở trên để chọn field và nhập giá trị",
        "placeholder": "Tìm kiếm..."
      }
    }
  }
}
```
